# 분류 앱 완성·배포 가이드

이 폴더는 완성 답안이 아니라 **실행 가능한 UI + 핵심 추론 TODO** 스캐폴딩입니다.

## 학생이 직접 완성할 코드

| TODO | 핵심 개념 | 완료 확인 |
|---|---|---|
| 1 | CNN feature extractor·분류부·forward | 체크포인트 구조와 일치 |
| 2 | `state_dict` 복원·`eval()` | 모델 로드 오류 없음 |
| 3 | 28×28 배열 → `(1,1,28,28)` float32 | 미리보기와 tensor 계약 일치 |
| 4 | `inference_mode`·softmax | 숫자 10개의 확률 표시 |

정답 코드는 주석에 제공하지 않습니다. 같은 과제 노트북의 `CNN`과 입력 shape을 근거로 작성하세요. 화면 레이아웃과 사진 정렬 보조 함수는 제공 범위입니다.

## 로컬 실행

노트북 [Step 7]을 먼저 실행해 이 폴더에 `mnist_cnn.pt`를 생성합니다.

```bash
cd 과제_Streamlit_앱_분류
python3.11 -m pip install -r requirements.txt
python3.11 -m streamlit run app.py
```

TODO를 아직 완성하지 않았으면 앱이 해당 번호를 알려줍니다. 번호 순서대로 수정하고 저장하면 Streamlit이 자동으로 다시 실행됩니다.

## 통합 showcase 권장

두 프로젝트를 모두 마친 뒤 상위 `미니프로젝트/` 폴더에서 실행합니다.

```bash
python3.11 -m streamlit run showcase/Home.py
```

최종 Cloud 배포는 `../showcase/README.md`의 통합 방식(Main file path `showcase/Home.py`)을 권장합니다. 분류 앱만 별도 배포하려면 Main file path를 `app.py`로 지정합니다.

`.env`, API key, 다른 개인 파일은 저장소에 올리지 않습니다.
